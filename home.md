# Welcome to the **ObsidianOS** Documentation

ObsidianOS is a UEFI‑only, systemd‑based operating system for **x86_64** systems, designed with an A/B partition layout for reliability.  
This documentation will guide you through installation, usage, and troubleshooting.

**Get started:**
- [Installation Guide](/install) — Step‑by‑step instructions for installing ObsidianOS
- [FAQs](/faqs) — Answers to common questions and issues.

> **Tip:** If this is your first time using ObsidianOS, start with the [Installation Guide](/install) and review the [System Requirements](/requirements) before you begin.
